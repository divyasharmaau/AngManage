About Manage

Manage is a Employee Management system made for learning based on Angular 11, Bootsrap 4 and Angular Material. It uses the custom Api's built in the Manage Employee Management App built using C# and ASP.Net Core MVC. It is used for maintaing various aspects of an employee profile.
It can be accesed by both the administrator and the employee and both will be presented with a different menu based on their role.
Manage has following systems:

  • Employee Management
  • Leave Management

Key Features

  • Role based access and visiblity of the menu and functionalities like add, edit or delete
  • Pagination and sorting for all the lists
  • Photo upload along with preview while uploading
  • Demo Admin and Demo User login
  • Abilty to reset password using forgot my password link in login

Functionality Access

The access to the different parts of the system is based on the role of the employee. While the Administrator has access to all parts of the system, the employee will be able to see only the parts that are relevant to them.

Administrator

Employee Management

  • List of all employees
  • Add New Employee: Add and edit employee details

Leave Management

  • List of all Leaves for all employees
  • Apply and edit New leave
  • Approve or Decline leave

Employee

Employee Management

  • Employee Profile
  • Add and edit Personal details

Leave Management

  • List of all their Leaves
  • Apply and edit New leave
  • Delete the leave which has not yet been approved or disapproved
